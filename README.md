## 🌟 Project Overview

**Patidar Enterprise** is a modern, professional web application showcasing complete solar energy solutions for residential and commercial customers across **Aravalli District and all of Gujarat, India**.

### What This Project Does

This website serves as a comprehensive digital presence for Patidar Enterprise, providing:

- 🏠 **Residential Solar Installation Services** - Rooftop solar systems for homes
- 🏢 **Commercial Solar Solutions** - Industrial and commercial installations
- 💧 **Solar Water Heaters** - Efficient water heating systems
- 🔧 **Maintenance & Support** - Cleaning, repair, and performance checks
- 📋 **Government Subsidy Assistance** - Help with eligibility and documentation
- ⚡ **Net Metering Support** - Grid connection coordination

### Service Area
- **Primary:** Aravalli District, Gujarat
- **Extended:** All of Gujarat, India

### 🔄 How Language Switching Works

1. **User clicks language button** → `data-lang-option` triggers
2. **applyLanguage() function executes:**
   - Updates DOM language attribute
   - Translates all text nodes using Map lookup
   - Translates all attributes (alt, aria-label, placeholder)
   - Updates page title and meta tags
   - Saves preference to localStorage
3. **Persistent storage** → Language preference survives page reloads

- Optimal reading width for text content

### 📱 Mobile Experience (<860px)
- Single-column layouts
- Full-width cards with appropriate padding
- Hamburger menu for navigation
- Touch-friendly button sizes (≥46px)
- Simplified grids (2-3 columns)
- Floating action buttons for quick contact

### 🎯 Key Responsive Features

```html
<!-- Flexible Images -->
<img> elements use 100% width and aspect-ratio preservation

<!-- Grid Adaptability -->
Services: 4 cols (desktop) → 3 cols (tablet) → 1 col (mobile)
Benefits: 7 cols (desktop) → 3 cols (tablet) → 1 col (mobile)
Gallery: 4 cols (desktop) → 2 cols (tablet) → 1 col (mobile)

<!-- Fluid Typography -->
h1: clamp(3rem, 7vw, 5.9rem) - scales with viewport
h2: clamp(2rem, 4vw, 3.25rem) - responsive sizing

<!-- Flexible Containers -->
```
